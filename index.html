Homework 3

Homework 3 was my most challenging. Are job was to create a Java web application that would estimate weekly net pay through a salary calculator. I learned how to create a basic .jsp page, learn the in and outs of netbeans and deploy a project to a Tomcat server so people could view it.

Homework 4

The basis of Homework 4 was building a Java web application that could read in a table from an Oracle database. I learned how to set up Oracle SQL Developer and connect it to Netbeans, create a database with sample data and utilize the MVC design pattern including servlets and different .jsp pages.

Homework 5

Homework 5 built upon what we did in Homework 4. I used my ability to add and delete records in my database from the application I created in homework 4. In this case the data I used was that of all of the current NFL Players. I learned how to use CSS in regards to the application to add more style, as well as how to add functionality through the MVC design pattern with servlets and queries.

Homework 6

In Homework 6 I added further functionality to my Java web application from Homework 4 and 5. I included search and update abilities for existing records in my database, as well as building a wireframe and other features for the application. I learned how to make a more appealing application and how to easily create a user interface for all the different parts of my application.
